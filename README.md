Yii2 User module
=========

Installation
------------

The preferred way to install this extension is through [composer](https://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist ser6-io/yii2-user "*"
```

or add

```
"ser6-io/yii2-user": "*"
```

to the require section of your `composer.json` file.
